# Attribute group

**Response**

| Key           | Type |Description  | Required |
|---------------|------|--------------|----------|
| id       | string      |  Attribute Id            | Yes      |
| label     |  string   |   Attribute Label           | Yes      |
| default     |  string   |   Attribute Code           | Yes      |




**Example**
```
{
    "id": "4977611b-4814-447d-9e2c-3020f5d7a3ea",
    "label": "Default",
    "default": true
  }
```
