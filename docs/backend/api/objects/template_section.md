# Template_section

**Response**

| Key       | Type     |Description    | Required |
|-----------|----------|---------------|----------|
| row       |    int   |  Number of row in which section title would be visible   | Yes      |
| title     |  string  |   Section title   | Yes      |


**Example**

```
   {
       "row": 1,
       "title": "Header 1"
     }
```
