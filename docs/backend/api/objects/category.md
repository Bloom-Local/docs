# Category

**Response**

| Key           | Type |Description  | Required |
|---------------|------|--------------|----------|
| id       | string      |  Category Id            | Yes      |
| name     |  [Translation](backend/api/objects/translation.md)   |   Category Code  | No |
| code     |  string   |   Category Type           | Yes      |

**Example**

```
{
  "id": "6843ca66-24c6-5901-8678-98e098f0b3ba",
  "name": {
    "PL": "spodnie"
  },
  "code": "code_6"
}
```
