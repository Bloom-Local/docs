#Draggable

## State

* #### **isListElementDragging**

    _State for is is list element dragging value_
    * default: **`false`**
    * type: **`Boolean`**

* #### **isColumnDragging**

    _State for is column dragging value_
    * default: **`false`**
    * type: **`Boolean`**

* #### **isFilterDragging**

    _State for is filter dragging value_
    * default: **`true`**
    * type: **`Boolean`**

* #### **draggedElement**

    _State for dragged element_
    * default: **`null`**
    * type: **`String`**
